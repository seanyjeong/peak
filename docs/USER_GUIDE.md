## 1. 시작하기 전에

### P-ACA에서 먼저 해야 할 일

P-EAK를 사용하려면 **P-ACA(학원 관리 프로그램)**에서 먼저 학생과 강사를 등록해야 합니다.

| 순서 | P-ACA에서 할 일 |
|:---:|----------------|
| 1 | 학생 등록하기 |
| 2 | 강사 등록하기 |
| 3 | 강사 출근 요일 설정하기 |

> 💡 **팁**: P-ACA에서 학생을 등록하면 P-EAK에서 자동으로 불러올 수 있어요!

## 3. 초기 설정 (원장님/관리자)

> ⚠️ **중요**: 이 설정은 처음 한 번만 하면 됩니다!

### 설정 순서

```
1. 측정 종목 만들기
        ↓
2. 배점표 만들기
        ↓
3. 운동 등록하기
        ↓
4. (선택) 운동 팩 만들기
```

---

### 3-1. 측정 종목 만들기

**실기측정설정** 메뉴에서 측정할 종목들을 등록합니다.

#### 종목 추가하는 방법

1. 왼쪽 메뉴에서 **실기측정설정** 클릭
2. **+ 종목 추가** 버튼 클릭
3. 종목 정보 입력:

| 입력 항목 | 설명 | 예시 |
|----------|-----|------|
| 종목명 | 측정할 운동 이름 | 제자리멀리뛰기 |
| 단위 | 측정 단위 | cm, 초, 개 |
| 기록 방향 | 좋은 기록 기준 | 높을수록 좋음 / 낮을수록 좋음 |

#### 기록 방향이란?

- **높을수록 좋음**: 숫자가 클수록 잘한 것
  - 예: 제자리멀리뛰기 (250cm > 200cm)
  - 예: 메디신볼던지기 (8m > 6m)

- **낮을수록 좋음**: 숫자가 작을수록 잘한 것
  - 예: 100m 달리기 (12초 < 15초)
  - 예: 왕복달리기 (45초 < 50초)

---

### 3-2. 배점표 만들기

학생 기록을 점수로 환산하는 기준표를 만듭니다.

#### 배점표 추가하는 방법

1. **실기측정설정** → **배점표** 탭 클릭
2. **+ 배점표 추가** 버튼 클릭
3. 정보 입력:

| 입력 항목 | 설명 | 예시 |
|----------|-----|------|
| 배점표 이름 | 알아보기 쉬운 이름 | 2026 서울대 체교 남자 |
| 대학교 | 해당 대학 | 서울대학교 |
| 학과 | 해당 학과 | 체육교육과 |
| 성별 | 남자/여자 | 남자 |
| 연도 | 적용 연도 | 2026 |

4. 종목별 점수 기준 입력:

예시) 제자리멀리뛰기 남자 배점:
| 기록 | 점수 |
|-----|-----|
| 280cm 이상 | 100점 |
| 270~279cm | 95점 |
| 260~269cm | 90점 |
| ... | ... |

---

### 3-3. 운동 등록하기

수업에서 사용할 운동들을 등록합니다.

#### 운동 추가하는 방법

1. 왼쪽 메뉴에서 **운동 관리** 클릭
2. **+ 운동 추가** 버튼 클릭
3. 운동 정보 입력:

| 입력 항목 | 설명 | 예시 |
|----------|-----|------|
| 운동명 | 운동 이름 | 박스점프 |
| 태그 | 운동 분류 | 하체, 순발력 등 |
| 세트/횟수 | 기본 세트와 횟수 | 3세트, 10회 |
| 설명 | 운동 방법 설명 | (선택사항) |

---

### 3-4. 운동 팩 시스템

운동 팩은 **여러 운동을 묶어서 저장**하고 **다른 학원과 공유**할 수 있는 기능입니다.

#### 운동 팩이 필요한 이유

- 우리 학원 운동 목록을 백업해둘 수 있어요
- 다른 학원에 우리 운동 세트를 공유할 수 있어요
- 다른 학원의 좋은 운동 세트를 가져올 수 있어요

#### 팩 만들기

1. **운동 관리** → **운동 팩** 탭 클릭
2. **+ 팩 만들기** 버튼 클릭
3. 팩 이름 입력 (예: "일산맥스 기본 운동세트")
4. 포함할 운동들 선택
5. **저장** 클릭

#### 팩 내보내기 (다른 학원에 공유하기)

1. 내보낼 팩의 **⋮** 메뉴 클릭
2. **내보내기** 선택
3. JSON 파일이 다운로드됨
4. 이 파일을 다른 학원에 전달

#### 팩 가져오기 (다른 학원 팩 받기)

1. **파일 가져오기** 버튼 클릭
2. 받은 JSON 파일 선택
3. 팩이 추가됨 (팩 이름 뒤에 "가져옴" 표시)

#### 팩 불러오기 (운동 목록 대체)

> ⚠️ **주의**: 이 기능은 현재 운동 목록을 **모두 지우고** 팩의 운동으로 바꿉니다!

1. 불러올 팩의 **⋮** 메뉴 클릭
2. **불러오기** 선택
3. 확인 메시지에서 **예** 클릭
4. 운동 목록이 팩의 운동들로 교체됨

## 4. 매일 하는 일

### 하루 일과 순서

```
출근 체크 → 반 배치 → 수업 계획 → 수업 진행 → 수업 기록 → 기록 측정
```

---

### 4-1. 출근 체크

오늘 출근한 강사를 확인합니다.

1. **출근 체크** 메뉴 클릭
2. 오늘 날짜 확인
3. 출근한 강사 체크 ✓

> P-ACA에서 설정한 요일에 자동으로 체크되어 있어요

---

### 4-2. 반 배치

오늘 수업할 학생들을 반별로 배치합니다.

#### 학생 배치하는 방법

1. **반 배치** 메뉴 클릭
2. 오늘 날짜와 시간대 선택
3. 왼쪽의 학생 목록에서 학생을 **끌어서** 원하는 반에 **놓기**

#### 강사 배치하는 방법

1. 반 이름 옆의 **강사 배치** 버튼 클릭
2. 담당 강사 선택
3. 주강사/보조강사 지정

#### 화면 설명

```
┌─────────────────────────────────────────────────────┐
│  [학생 목록]          │  [1반]    [2반]    [3반]   │
│                       │                            │
│  김철수               │  학생A    학생D    학생G   │
│  이영희    ──────────>│  학생B    학생E    학생H   │
│  박지민               │  학생C    학생F    학생I   │
│  ...                  │                            │
│                       │  강사: 홍길동  강사: 김강사│
└─────────────────────────────────────────────────────┘
     학생을 끌어서 ────────> 반에 놓으세요
```

> 💡 **팁**: 학생을 다른 반으로 옮기려면 그냥 끌어서 옮기면 돼요!

---

### 4-3. 수업 계획

오늘 할 운동을 미리 계획합니다.

#### 수업 계획 작성 방법

1. **수업 계획** 메뉴 클릭
2. 날짜와 시간대 선택
3. 반 선택
4. **+ 운동 추가** 버튼으로 오늘 할 운동 추가

#### 환경 체크

수업 전 체크할 항목들:
- ☐ 장비 점검 완료
- ☐ 청소 상태 확인
- ☐ 냉난방 확인

---

### 4-4. 수업 기록

수업 중 학생별 상태를 기록합니다.

#### 기록하는 방법

1. **수업 기록** 메뉴 클릭
2. 날짜, 시간대, 반 선택
3. 학생별로 기록:

| 기록 항목 | 설명 |
|----------|-----|
| 컨디션 | 😊 좋음 / 😐 보통 / 😫 나쁨 |
| 메모 | 특이사항 자유롭게 작성 |

> 💡 **팁**: 부상이나 특이사항은 꼭 메모해두세요!

---

### 4-5. 기록 측정

학생들의 실기 기록을 측정하고 저장합니다.

#### 기록 입력 방법

1. **기록 측정** 메뉴 클릭
2. 날짜 선택
3. 측정할 종목 선택
4. 학생별 기록 입력
5. **저장** 버튼 클릭

#### 기록 입력 화면

```
┌────────────────────────────────────────┐
│  종목: 제자리멀리뛰기        2026.01.14│
├────────────────────────────────────────┤
│  학생명       기록      이전최고  향상  │
│  ─────────────────────────────────────│
│  김철수      [255  ]cm   250cm   +5   │
│  이영희      [238  ]cm   235cm   +3   │
│  박지민      [242  ]cm   245cm   -3   │
└────────────────────────────────────────┘
```

> 💡 **팁**:
> - 하루에 같은 종목을 여러 번 측정해도 **가장 좋은 기록**만 저장돼요
> - 이전 최고 기록과 비교해서 향상도를 바로 확인할 수 있어요

## 5. 월말테스트

매달 말에 진행하는 정기 테스트를 관리합니다.

### 5-1. 테스트 만들기

1. **월말테스트** 메뉴 클릭
2. **+ 새 테스트** 버튼 클릭
3. 테스트 정보 입력:

| 입력 항목 | 설명 | 예시 |
|----------|-----|------|
| 테스트 이름 | 알아보기 쉬운 이름 | 2026년 1월 월말테스트 |
| 테스트 날짜 | 진행 날짜 | 2026-01-31 |

---

### 5-2. 세션 추가하기

> **세션**이란? 테스트를 나눠서 진행할 때 각각의 시간대를 말해요.
> 예: 오전 세션, 오후 세션

1. 테스트 상세 화면에서 **+ 세션 추가** 클릭
2. 세션 정보 입력:
   - 세션 이름 (예: 1차 오전)
   - 시작 시간

---

### 5-3. 참가자 등록하기

#### 재원생 추가

학원에 등록된 학생을 테스트에 추가합니다.

1. **참가자 관리** 탭 클릭
2. **재원생 추가** 버튼 클릭
3. 참가할 학생 선택
4. 참가할 세션 선택

#### 외부 응시자 추가

학원에 등록되지 않은 외부인을 추가합니다.

1. **외부 응시자 추가** 버튼 클릭
2. 정보 입력:

| 입력 항목 | 필수 |
|----------|-----|
| 이름 | ✓ |
| 연락처 | ✓ |
| 성별 | ✓ |
| 학교/학년 | |

---

### 5-4. 조 편성하기

참가자들을 조별로 나눕니다.

1. **조 편성** 탭 클릭
2. 자동 편성 또는 수동 편성 선택
3. 참가자를 끌어서 조에 배치

---

### 5-5. 테스트 진행하기

#### 테스트 상태 변경

| 상태 | 설명 |
|-----|------|
| 준비 | 아직 시작 전 |
| 진행 중 | 현재 테스트 중 (기록 입력 가능) |
| 완료 | 테스트 종료 |

1. 세션의 상태를 **진행 중**으로 변경
2. 종목별로 기록 입력
3. 모든 기록 입력 후 **완료**로 변경

#### 기록 입력

1. 세션 클릭하여 상세 화면 진입
2. 종목 탭 선택
3. 참가자별 기록 입력
4. **저장** 클릭

---

### 5-6. 결과 내보내기

테스트 결과를 엑셀 파일로 저장할 수 있습니다.

1. 테스트 상세 화면에서 **📥 엑셀 다운로드** 버튼 클릭
2. 파일이 자동으로 다운로드됨

> 파일명 예시: `2026년_1월_월말테스트_기록.xlsx`

---

### 6-2. 학생 상세 정보

학생을 클릭하면 다음 정보를 볼 수 있어요:

#### 기본 정보
- 이름, 학교, 학년
- 목표 대학/학과
- 등록 상태

#### 기록 그래프

```
점수
 │
100├─────────────────────●
 90├───────────●────●───
 80├─────●────
 70├────
    └─────────────────────────→ 날짜
      1월    2월    3월    4월
```

- 종목별 기록 변화를 그래프로 확인

- 향상 추이 한눈에 파악

#### 종목별 기록
- 각 종목의 최고 기록
- 최근 측정 기록
- 목표 기록 대비 달성률

### 6-3. 대시보드

- 오늘 출석 현황
- 최근 측정 기록
- 학생별 향상도 순위
- 이번 달 수업 현황

## 7. 자주 묻는 질문

### Q. 학생이 안 보여요

**A.** P-ACA에서 학생이 등록되어 있는지 확인해주세요.
1. P-ACA에서 학생 등록 확인
2. P-EAK **학생 관리** → **동기화** 버튼 클릭

---

### Q. 강사가 안 보여요

**A.** P-ACA에서 강사 등록과 출근 요일을 확인해주세요.
1. P-ACA에서 강사 등록 확인
2. P-ACA에서 해당 요일 출근 설정 확인

---

### Q. 기록을 잘못 입력했어요

**A.** 같은 날 같은 종목을 다시 입력하면 됩니다.
- 더 좋은 기록이면 자동으로 업데이트
- 더 나쁜 기록이면 이전 기록 유지

---

### Q. 배점표가 적용이 안 돼요

**A.** 배점표 설정을 확인해주세요.
1. 학생의 성별과 배점표 성별이 같은지 확인
2. 학생의 목표 대학/학과와 배점표가 연결되어 있는지 확인

---

### Q. 태블릿에서 화면이 이상해요

**A.** 태블릿을 가로로 눕혀보세요.
- 가로 모드: 더 넓은 화면
- 세로 모드: 모바일 스타일 화면

### Q. 월말테스트 기록이 안 보여요

**A.** 세션 상태를 확인해주세요.
1. 세션이 **완료** 상태인지 확인
2. 기록을 입력한 후 **저장** 버튼을 눌렀는지 확인

### Q. 운동 팩을 불러왔는데 기존 운동이 사라졌어요

**A.** 정상입니다. "팩 불러오기"는 기존 운동을 모두 지우고 팩의 운동으로 교체하는 기능이에요.
- 기존 운동을 유지하면서 추가하고 싶으면 **팩 가져오기** 후 수동으로 추가해주세요
